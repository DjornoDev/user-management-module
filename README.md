## ğŸ› ï¸ Development Quick-Start

### System Requirements
- PHP 8.1 or higher
- Composer 2.5+
- MySQL 8.0+
- Node.js 18+

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-org/bit-admission-portal.git
   cd bit-admission-portal
   ```

2. **Install dependencies**
   ```bash
   composer install
   npm install
   ```

3. **Set up environment**
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

4. **Configure your database in .env**
   ```
   DB_CONNECTION=mysql
   DB_HOST=127.0.0.1
   DB_PORT=3306
   DB_DATABASE=user_management_module
   DB_USERNAME=root
   DB_PASSWORD=
   ```

5. **Launch development server**
   ```bash
   php artisan serve
   npm run dev
   ```
